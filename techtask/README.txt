Техническое задание было выполнено для отбора в школу NFactorial. В файле представлен класс UserManager, который позволяет пользоваться системой учета пользователей.
Функционал:
addUser( string ) - позволяет добавить пользователя в систему учета. Пользователю присвается уникальный идентификатор для дальнейшего использования. Возвращает уникальный идентификатор. Пр: “””print( userManager.addUser(“Toqtar”))””” - возвращает 1 - идентификатор добавленного пользователя.
getUser( int ) - позволяет получить имя пользователя по присвоенному ему идентификатору. Пр: “””print(userManager.getUser(1))””” - вернет “Toqtar”.
deleteUser( int ) - позволяет удалить пользователя из системы учета пользователя по егоидентификатору. При удачном выполнении действия возвращает “true”, и при неудачном - “false”. Пр: “””print(userManager.deleteUser(1))””” - вернет “true” и удалит пользователя из системы.
findUserByName( string ) - позволяет получить идентификаторы всех пользователей системы по имени. Пр: “””print(userManager.findUserByName(“Toqtar”))””” - ничего не выведет так как такие пользователи отсутствуют.
